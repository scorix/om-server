# 要素清单快照

与 [`../open-meteo-data.md`](../open-meteo-data.md) 配套。

| 文件 | 说明 |
|------|------|
| `{model}-spatial.txt` | `data_spatial/{model}/latest.json` → `variables` |
| `{model}-run.txt` | `data_run/{model}/latest.json` → `variables` |
| `{model}-timeseries.txt` | `data/{model}/` 下已发布变量目录 |
| `CATALOG.md` | 按气象类别分组 |
| `manifest.json` | 快照元数据 |

模型：`ecmwf_ifs025`、`ncep_gfs025`、`dwd_icon`。
